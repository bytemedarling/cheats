How to Create a Table

Tag Name Description
<table> table wrapper element for all HTML tables
<thead> table head set of rows defining the column headers in a table
<tbody> table body set of rowss containing acctual table data
<tr> table row table row container (row across table)
<td> table data what you want inside the row
<tfoot> table foot set of rows defining the footer in a table

Table Attributes

Attribute Name Description
colspan column span defines how many columns a td element should
rowspan row span defines how many rows a td element should span
